Phân Quyền

Theo quyền sở hữu (Ownership)

Mỗi file/foler đều được gán 3 kiểu user:

  • Owner: mặc định file/folder được tạo ra bởi user nào thì file/folder đó thuộc quyền sở hữu của user đó (để biết user hiện tại sử dụng command whoami hoặc kiểm tra đầu dòng của terminal)
  • Group: tập hợp 1 nhóm user (kiểm tra user hiện tại có thuộc group không bằng cách sử dụng command groups)
  • Other: tất cả những user còn lại

Sử dụng command chown để chuyển quyền sở hữu sang user/group khác.

Theo quyền thực thi (Permissions)

Mỗi file/folder sẽ có 3 quyền thực thi dành cho từng kiểu user:

  • Read: viết tắt là r và tượng trưng cho số 4, quyền đọc
    • Nếu là file thì user sẽ đọc được nội dung
    • Nếu là folder thì user sẽ xem được danh sách file/folder bên trong
  • Write: viết tắt là w và tượng trưng cho số 2, quyền viết
    • Nếu là file thì user sẽ chỉnh sửa file đó
    • Nếu là folder thì user có thể thêm, xóa và đổi tên các file/folder trong thư mục đó.
  • Execute: viết tắt là x và tượng trưng cho số 1, quyền chạy file (tương tự file .exe trên Windows)
    • Nếu là file thì user có thể khởi chạy file
    • Nếu là folder thì user có thể truy cập folder đó (command cd)

Sử dụng command chmod để thay đổi quyền thực thi.

Thực hành

Để xem quyền, thường sử dụng command ls -l

Theo như ảnh trên:

  • Folder app:

    • Owner (sshuser) có full quyền: read, write, execute | rwx | 4 + 2 +1 =7
    • Group (www-data) và Other đề có 2 quyền read, execute | r-x | 4 + 0 + 1 = 5 ==> folder app có có quyền thực thi là 755

  • File auth.json:

    • Owner (sshuser) có 1 quyền: read | r-- | 4 + 0 + 0 = 4
    • Group (www-data) và Other không có quyền | --- | 0 + 0 + 0 = 0 ==> file auth.json có có quyền thực thi là 400

  • File auth.json.sample:

    • Owner (sshuser) có 2 quyền: read, write | rw- | 4 + 2 + 0 = 6
    • Group (www-data) và Other có 1 quyền read | r-- | 4 + 0 + 0 = 4 ==> file auth.json.sample có có quyền thực thi là 644

Ngoài dạng số (r-4, w-2, x-1) chúng ta cũng có thể sử dụng dạng chữ để phân quyền thực thi qua chmod:

Tên file/folder Quyền thực thi Câu lệnh sử dụng dạng số Câu lệnh sử dụng dạng chữ
app rwxr-xr-x chmod 755 chmod u+rwx g+rx o+rx
auth.json r——– chmod 400 chmod u+r
auth.json.sample rw-r–r– chmod 644 chmod u+rw g+r o+r
